# DAYMAP

A daily life organizer that combines task management, map visualization, 
driving route optimization and even public transit planning in one interface

This project is built for the 2026 Congressional App Challenge, competing in 
Maryland's 2nd Congressional District

## Tech Stack:
-Frontend: HTML5, CSS, JavaScript
-Map API's: Leaflet.js + OpenStreetMap
-Routing API: OSRM
-Location Search API: Nominatim
-Transit API: Navitia
-BackendD: Java Spring Boot
